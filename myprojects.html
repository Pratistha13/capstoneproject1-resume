PROJECTS

  • Student Tutor Application (August 2020)

  • Developed a backend for students/tutors which aim to connect students with tutors to learn the subjects they want to master on.
  • I implemented signup sign-in features, CRUD functionalities for add, read, update their information.
  • I added the send email functions using Java mail Sender and mime message. Users can also attach files, which was used as a Multipart file and then linked to the PostgreSQL database.
  • Application was deployed to Heroku and available for users. Username/password: user/user: admin/admin

  • Personal Trainer Web App Front-end (August 2020)

  • Developed a front-end application using React JS for a company to allow users to reserve personal trainer session.
  • REST API implementation/documentation was used to create a frontend application.
  • Application allows adding, deleting and updating clients personal data. Personal training sessions could be entered for each customer.
  • Calendar application was created to provide the statistics of the user activities.
  • Frontend application was built using designs defined in Material UI, Lodash, react-big-calendar, recharts.

  • Athletes and Sponsors matching Mobile Application (August 2020)

  • Developed a mobile app which matches athletes to sponsors and viceversa.
  • Swiper feature was implemented along with login and signup.
  • Application allowed message to be sent between sponsors and athletes. Authentication was done using firebase.

  • Database Application as a case assignment (December 2019)

  • Developed a database application as a course assignment which helped event organizers to organize concerts, dance and theatre performances and other cultural events.
  • Clients could book, update, and cancel tickets
  • Application also allowed organizers to remove non purchased tickets.
  • Validated the design of the database and test the implementation of the database queries.
